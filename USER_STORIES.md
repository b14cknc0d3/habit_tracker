# User Stories

1.  **User Registration:** As a new user, I want to be able to create an account so that I can track my habits.
2.  **User Login:** As a registered user, I want to be able to log in to my account to access my habit data.
3.  **Habit Creation:** As a logged-in user, I want to be able to create new habits to track.
4.  **Habit Tracking:** As a logged-in user, I want to be able to mark my habits as complete for the day.
5.  **Habit Visualization:** As a logged-in user, I want to be able to see my progress for each habit over time.
6.  **Habit Details:** As a logged-in user, I want to be able to view the details of a specific habit.
7.  **Favorites:** As a logged-in user, I want to be able to mark certain habits as favorites for easy access.
8.  **Settings:** As a logged-in user, I want to be able to access a settings screen to configure the app.
9.  **Notifications:** As a logged-in user, I want to be able to receive reminders to complete my habits.
